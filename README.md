# 🕵️‍♂️ 1A2B 邏輯小偵探訓練營 (1A2B Logic Trainer)

這是一個專門為孩子設計的 **1A2B 猜數字邏輯訓練遊戲**。使用 Python (Flask) 作為邏輯核心，搭配現代化、色彩繽紛且互動性極強的 Web 介面。

本程式不單單是一個簡單的猜數字遊戲，更是一個**邏輯推理的輔助教學工具**！

---

## 🌟 特色功能 (Features)

### 1. 🕵️‍♂️ 即時偵探分析 (Smart Detective Helper)
* 當孩子輸入完 4 個數字但**尚未送出**之前，後台的「小貓頭鷹偵探」會自動比對之前的猜測歷史。
* 如果目前輸入的數字**違反了之前的任何一條線索**，偵探會立刻指出矛盾（例如：「*如果答案是 5690，那你的第 1 次猜測 5678 應該會是 2A 而不是 0A 喔！*」），教導孩子系統性地檢查自己的假設是否合理，培養嚴謹的邏輯思維。

### 2. 🧠 邏輯推理看板 (Deduction Board)
* **數字狀態燈**：自動將 0-9 數字標記為三個狀態：
  * 🔴 **排除 (Eliminated)**：邏輯上絕對不可能出現在謎底中的數字。
  * 🟢 **包含 (Confirmed)**：邏輯上絕對存在於謎底中的數字。
  * ⚪ **可能 (Possible)**：目前仍不確定的數字。
* **位置鎖定器 (Position Locks)**：當系統推導出某個位置的數字是固定值時，會自動在看板上顯示該位置的數字，幫助孩子具體化消去法的成果。
* **剩餘可能組合數**：即時顯示目前符合所有線索的答案剩餘多少種組合（從一開始的 5,040 種逐步減少），讓孩子感受到每一次邏輯排除所帶來的巨大進展！

### 3. 🎵 動態互動與音效
* 內建互動式小鍵盤，支援實體鍵盤與滑鼠點擊。
* 採用 Web Audio API 動態合成的純淨小音效（無需下載任何音訊檔），包含：按鍵點擊聲、邏輯衝突警告聲、猜對提示音、以及獲勝時的歡慶交響樂。

## 🛠️ 安裝與啟動說明 (Installation & Run)

本程式不需要安裝複雜的依賴庫，會自動檢測並安裝需要的套件。

### 步驟 1：確保電腦已安裝 Python 3
請在終端機 (Terminal) 中確認 Python 是否已安裝：
```bash
python --version
# 或
python3 --version
```

### 步驟 2：啟動遊戲
在專案根目錄下，直接執行 `app.py`：
```bash
python app.py
# 或
python3 app.py
```
* **注意**：程式會自動檢查是否安裝了 `Flask`。如果沒有，它會**自動嘗試使用 pip 安裝**。
* 伺服器啟動後，程式會**自動在您的瀏覽器中開啟** `http://127.0.0.1:5000` 遊戲網頁，即可開始遊玩！

## 🎮 1A2B 遊戲規則科普

1. 系統會秘密生成一個由 **4 個不重複數字** (0-9) 組成的密碼（例如 `1234`，支援首位為 0，共有 5040 種組合）。
2. 玩家每次猜測一個 4 位數：
   * **A** 代表：數字正確，且**位置也正確**。
   * **B** 代表：數字正確，但**位置不正確**。
3. 舉例：若謎底為 `1234`，玩家猜 `1356`：
   * `1` 數字與位置皆對 (1A)。
   * `3` 數字對但位置錯 (1B)。
   * `5`、`6` 不在謎底中。
   * 系統回饋為 `1A1B`。
4. 當玩家得到 `4A0B` 時，代表 4 個數字和位置都完全正確，挑戰成功！
